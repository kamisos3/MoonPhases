# Moon Phase Tracker

▶ **Introduction**

A simple Moon Phase Tracker built with a React front-end and Python back-end. This web application provides real-time moon phase calculations, visual representations, and basic lunar information. Created to help users track lunar cycles and design front-end using JavaScript, CSS and HTML with React.

▶ **Table of Contents**

- [Overview](#overview)
- [Concepts](#concepts)
- [Installation](#installation)
- [Author](#author)

▶ **Overview**

The Moon Phase Tracker is a full-stack application that calculates and displays current moon phases with precision. It features an intuitive React-based user interface that presents moon phase data in a visually appealing format, with a Python backend using astronomical calculations.

▶ **Concepts**

- Real-time lunar phase calculations
- React component-based frontend architecture
- Python astronomical computations
- Responsive web design
- RESTful API integration
- Swiss Ephemeris for accurate astronomical data
